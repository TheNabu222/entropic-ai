## Deployment to Neocities

### Method 1: Manual Upload via Web Interface

1. Log in to your Neocities account at https://neocities.org
2. Navigate to your site dashboard
3. Create a new folder called `flux` (or any name you prefer)
4. Upload all three files (`index.html`, `style.css`, `script.js`) to that folder
5. Access the page at `https://coaiexist.wtf/flux/` (or your chosen folder name)

## Technical Details

### Features

- **CRT Effect**: Scanline overlay with subtle flicker animation
- **Glitch Effect**: Text glitching on the main title
- **Terminal Aesthetic**: Styled like a Unix terminal window
- **Real-time Clock**: Updates every second in CDT timezone
- **Responsive Design**: Works on mobile and desktop
- **Accessibility**: Respects `prefers-reduced-motion` for users who need it
- **Easter Egg**: Konami code reveals a hidden console message

### Browser Compatibility

- Works in all modern browsers (Chrome, Firefox, Safari, Edge)
- No external dependencies - pure HTML/CSS/JS
- No tracking, no analytics, no external resources
- Fully static - perfect for Neocities hosting

### Customization

If you want to modify the colors, edit the CSS variables at the top of `style.css`:

```css
:root {
    --primary-green: #00ff41;
    --secondary-cyan: #00ffff;
    --warning-red: #ff0040;
    --bg-black: #0a0a0a;
    --terminal-bg: #0d1117;
    --text-white: #e6e6e6;
    --text-dim: #8b949e;
}
```
