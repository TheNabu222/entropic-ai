üöÄ Getting Started

  • Copy the 3 lines of code from your chosen theme
  • Paste them in the <head> section of your HTML page
  • Edit config.js to customize your navigation links
  • Done! The navbar will automatically appear at the bottom

üé® Create Your Own Theme

  • Copy the template folder: cp -r themes/template themes/my-theme
  • Edit nav.css to change colors, fonts, and styles
  • Update config.js with your navigation structure
  • CSS variables make customization super easy!
